## 概要

* Java（Spring Boot）を主軸としたバックエンド開発を得意とし、業務系システムを中心にWebアプリケーション・API・バッチ処理の設計から実装・テストまで一貫して担当してきました。
* PHP（Laravel）・Vue.jsの開発経験もあり、バックエンドからフロントエンドまで幅広く対応可能です。また AWS（S3・EC2等）を用いたクラウド環境での開発経験もあります。
* 基本設計・詳細設計から実装、テストまでの一連の工程を継続して担当しており、ウォーターフォール／スクラム双方の開発方式でプロジェクトに従事してきました。
* サブリーダーの経験もあり、コードレビュー・進行管理・顧客折衝を担いながらチームの品質向上と円滑なプロジェクト推進に貢献してきました。
